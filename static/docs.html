使用指南

SubHub 是一个运行在 Cloudflare Workers 上的订阅管理面板，登录后即可查看你的机场订阅信息。

用户登录

  1. 访问首页，输入你的订阅 UUID。
  2. 登录成功后即可查看订阅链接、到期时间及流量用量。
  3. 点击“刷新”按钮即可实时更新订阅 header 的流量数据。

管理员操作

使用管理员口令登录后，你可以：

  • 添加新的订阅用户。
  • 修改用户的订阅链接、到期时间、备注及流量基数。
  • 删除无效用户或过期订阅。

常见问题

  • 订阅 URL 需可在 Cloudflare 网络中访问，否则刷新功能会失败。
  • JWT 登录态默认 24 小时，如需更长时间可调整环境变量。
  • KV 数据结构固定，如需扩展可以在源码中新增字段。
